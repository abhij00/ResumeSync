
# ResumeSync

ResumeSync is a simple resume management and candidate ranking system that helps recruiters upload resumes, create jobs roles, and view ranked candidates.

## Installation

Clone the repository

```
git clone https://github.com/abhij00/ResumeSync.git
cd ResumeSync
```

---

## Backend Setup

Go to backend folder

```
cd backend
```

Create virtual environment

```
python -m venv venv
```

Activate environment (Windows)

```
venv\Scripts\activate
```

Install dependencies

```
pip install -r requirements.txt
```

Run backend

```
python app.py
```

Backend runs on:

```
http://localhost:5000
```

---

## Frontend Setup

Open another terminal

```
Go to frontend folder
```

Install dependencies

```
npm install
```

Run frontend

```
npm run dev
```

Frontend runs on:

```
http://localhost:5173
```
